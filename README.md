An interactive browser-based Hangman game built using HTML, CSS, and JavaScript. Players guess hidden words from multiple categories while tracking their progress through a dynamic hangman illustration. The game includes difficulty levels, hints, score tracking, and persistent statistics using Local Storage.

## 🎮 Gameplay

1. Select a difficulty level.
2. A random word is chosen from the selected category.
3. Guess letters using the on-screen keyboard.
4. Correct guesses reveal letters in the word.
5. Incorrect guesses draw parts of the hangman.
6. Win by revealing the complete word before reaching 6 wrong attempts.
7. Use hints when needed.

---

## 📈 Statistics

The game automatically stores:

* Total Wins
* Total Losses

using Local Storage, allowing players to retain progress between sessions.
